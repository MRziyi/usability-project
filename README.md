## What’s in this website?
The site documents our end-to-end process across phases:
- **3A** Group Organization (plan, tooling, billing system)
- **3B** User Study
- **3C** First Redesigns
- **3D** Redesign Comments (peer feedback received)
- **3E** Group Reply (our responses/actions)
- **3F** Final Redesigns

It is built to be **content-driven**: most updates should be done by editing **Markdown/MDX** under `/content` — no code changes needed.
