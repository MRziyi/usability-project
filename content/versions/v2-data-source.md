## Problem Statement

Connecting a new data source in Looker Studio is confusing for first-time users. The multi-step wizard uses inconsistent terminology and lacks clear progress indicators.

## Design Changes

- Redesigned connection wizard with clear step indicators
- Simplified data source selection with search and categories
- Added inline previews of data before committing the connection
- Improved error messaging for failed connections
