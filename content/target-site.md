## What is Google Looker Studio?

Google Looker Studio (formerly Google Data Studio) is a free, web-based data visualization and reporting tool that allows users to create interactive dashboards and reports from various data sources. It is part of Google's business intelligence suite.

## Typical User Goals

- Create visual reports and dashboards from data sources
- Connect to and manage multiple data sources (Google Sheets, BigQuery, etc.)
- Share reports with stakeholders and collaborators
- Customize chart types, styling, and interactivity
- Schedule automated report delivery

## Key Workflows (Evaluation Scope)

Our evaluation focuses on the **report creation and editing flow**, specifically:

1. **Create a new report** — Starting from scratch, selecting a template, or duplicating
2. **Connect data sources** — Adding, configuring, and managing data connections
3. **Edit charts and visualizations** — Configuring chart types, dimensions, metrics, and styles
4. **Share and publish** — Sharing reports with specific users or publicly

## Out of Scope

- Admin and billing settings
- BigQuery-specific features
- Mobile viewing experience (we focus on the desktop editing interface)
- API integrations
